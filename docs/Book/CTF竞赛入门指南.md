# CTF竞赛入门指南

强烈推荐！入门指南！

本书主要面向CTF初学者，专注于Linux二进制安全。全书包含12章，从二进制底层讲起，结合源码详细分析了常见二进制安全漏洞、缓解机制以及漏洞利用方法，并辅以分析工具和环境搭建的讲解，循序渐进，让读者可以进行系统性的学习。本书在内容和素材的选择上较为连续完整，每个知识点都配以经典例题，并花费了大量篇幅进行讲解，旨在最大程度地还原分析思路和解题过程，达到举一反三的效果。
